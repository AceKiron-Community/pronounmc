# PronounMC
PronounMC is a Minecraft plugin designed to make it easier to keep track of pronouns.

## Supported pronouns
* Unspecified
* Any
* Other
* Ask
* Avoid
* He/him
* He/she
* He/they
* It/him
* It/its
* It/she
* It/they
* She/he
* She/her
* She/it
* She/they
* They/he
* They/it
* They/she
* They/them
